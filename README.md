# ui-parts
## 概要
JavaScriptで実装したUIパーツ集です。

実務でよく使用されるUI（アコーディオン・モーダル・タブ・トースト通知・コピー機能など）を、
アクセシビリティと可読性を意識して実装しました。

---

## 制作目的
フロントエンド実装力の向上と、
「実務で使えるJavaScriptコード」を意識したポートフォリオ制作を目的としています。

UIの挙動だけでなく、
構造設計・状態管理・再利用性も意識しています。

---

## 使用技術

- HTML5
- CSS3
- JavaScript（ES6）
- DOM操作
- イベントリスナー
- aria属性によるアクセシビリティ対応

---

## 実装パーツ一覧

### ① Accordion（アコーディオン）
- クリックで開閉
- aria-expanded管理
- hidden属性で表示制御

### ② Modal（モーダル）
- 背景クリックで閉じる
- ESCキー対応
- フォーカストラップ実装
- aria-modal対応

### ③ Tabs（タブ切り替え）
- aria-selected制御
- hidden属性でパネル管理
- 状態の明確な分離

### ④ Toast通知
- 自動消滅
- 複数表示対応
- aria-liveによる通知

### ⑤ Copy機能
- Clipboard API使用
- 成功/失敗のフィードバック表示

---

## 工夫した点

### 状態管理の明確化
UIの開閉状態や選択状態を属性で管理し、
DOM構造とロジックの分離を意識しました。

### アクセシビリティ配慮
aria属性やキーボード操作への対応を取り入れ、
ユーザー体験の向上を意識しています。

### 再利用性を考慮
data属性を用いて、複数設置できる設計にしています。
